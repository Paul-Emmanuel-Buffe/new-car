# New Car: Price Prediction Project

This project is a data science application focused on predicting vehicle prices using Linear Regression. It is designed as an educational exercise for a French-speaking audience.

## Project Overview
The goal of this project is to use data from the CarDekho website to determine the fair market value of a car. By analyzing various characteristics of a vehicle, we can predict its price to help buyers make informed decisions and ensure they get a good deal.

## Key Objectives
* Data Exploration: Loading the dataset and calculating basic statistics such as the average, median, and quartiles.
* Visualization: Creating charts to see how the data is distributed and how different features relate to each other.
* Statistical Analysis: Measuring the strength of the relationship between a car's age and its selling price using the Pearson correlation.
* Machine Learning: Building mathematical models to predict a price based on factors like distance traveled and the type of transmission.
* Practical Case: Estimating a recommended price for a specific request, such as a vehicle less than 7 years old with less than 100,000 km.

## Technical Tools
* Language: Python
* Main Libraries: Pandas, Scipy, Scikit-Learn, Numpy, Matplotlib, and Seaborn
* Platform: Jupyter Notebook

## Learning Outcomes
This project demonstrates the ability to:
* Extract and aggregate data from different sources.
* Design the technical framework for an application using artificial intelligence.
* Solve technical issues to ensure the application works correctly.
* Explain technical results in a way that is useful for the end user.
